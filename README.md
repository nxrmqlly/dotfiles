## Prerequisites
1. [GNU Stow](https://www.gnu.org/software/stow/)
2. [zsh](https://www.zsh.org/)

## Installation
1. Clone the repository
```
git clone https://github.com/nxrmqlly/dotfiles.git ~/dotfiles
```
2. Use stow to symlink the files
```
cd ~/dotfiles
stow .
```
